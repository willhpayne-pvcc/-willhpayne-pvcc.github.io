Operating System Termonology
Term Definition
Operating System The software that controls the operation of the computer and includes file-management, the UI, and program execution.
Job queue A job is the execution of a single program, and a queue is a method of organization based on First In First Out(FIFO). A job queue is a list of jobs to be done based on FIFO order.
FIFO First In First Out is a method of organizing job execution.
Interactive processing When a user inputs something into a computer and recieves an output immediatly.
Timesharing A computer executing processes for many people at one time.
Multi-processing A computer executing many processes for one user.
Application software A program that is designed to carry out tasks unrelated to the computers core functions.
System software System software creates a platform for a computer to run other software; and OS is an example of system software.
Utility software designed to run diagnostics, maintain the core operations, and aqquire data on how the computer is running.
GUI Is part of the User Interface. Uses graphical representations of computer processes to increase ease of use.
Shell An older form of UI that used a command window that one could type text prompts in to.
Kernel The core of the OS and the part the user does not directly interact with.
Device driver Communicates with controllers to use peripheral devices.
File manager Part of the kernal of the OS, and it records the location of files, what users can accsess those files, and what space is free.
Memory manager Part of the kernal; regulates the usage of main memory.
Virtual memory The memory created on a mass-storage unit via a process called paging. Paging is used when memory needs exceed what is provided in the main memory. The machine stores the data in mass-storage in chunks called pages which then get moved between main mem and storage as needed.
boot strapping The process of booting up the OS of the machine.
ROM Read Only Memory stores data that can not be altered after the units manufacture.
Deadlock When a process is incapable of being completed due to it waiting on another process or prompt from a user to continue.
Spooling The process of storing data to be used by an output device.
Administrator A user with complete accsess to all aspects of a machine.
Multi-factor authentication A security mechanism that requires a user to provide one or more pieces of evidence to sign in.
